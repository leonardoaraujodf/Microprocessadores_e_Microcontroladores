1) O que são sistemas embarcados?

Um sistema embarcado é um "computador" que possui uma função dedicada dentro de um dispositivo. Este é embarcado (ou seja, interno) como uma parte de um dispositivo completo. Os sistemas embarcados possuem microprocessadores, memórias, clocks, entradas ETHERNET etc. Os sistemas embarcados são comumente encontrados na cozinha, indústria, automóveis, comércio, aplicações militares, saúde, etc.
